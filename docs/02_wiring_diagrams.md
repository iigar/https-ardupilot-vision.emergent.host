## 2. Підключення Raspberry Pi Zero 2 W до Matek FC (UART)

### Розпіновка Pi Zero 2 W (GPIO Header)

```
                    ┌─────────────────────────────┐
                    │      Pi Zero 2 W (зверху)   │
                    │                             │
   3.3V  (1) ●──────┼─────────────────────────────┼──────● (2)  5V
  GPIO2  (3) ○──────┼─────────────────────────────┼──────○ (4)  5V
  GPIO3  (5) ○──────┼─────────────────────────────┼──────○ (6)  GND ◄── до Matek GND
  GPIO4  (7) ○──────┼─────────────────────────────┼──────● (8)  TXD ◄── до Matek RX
    GND  (9) ○──────┼─────────────────────────────┼──────● (10) RXD ◄── до Matek TX
 GPIO17 (11) ○──────┼─────────────────────────────┼──────○ (12) GPIO18
 GPIO27 (13) ○──────┼─────────────────────────────┼──────○ (14) GND
 GPIO22 (15) ○──────┼─────────────────────────────┼──────○ (16) GPIO23
   3.3V (17) ○──────┼─────────────────────────────┼──────○ (18) GPIO24
 GPIO10 (19) ○──────┼─────────────────────────────┼──────○ (20) GND
  GPIO9 (21) ○──────┼─────────────────────────────┼──────○ (22) GPIO25
 GPIO11 (23) ○──────┼─────────────────────────────┼──────○ (24) GPIO8
    GND (25) ○──────┼─────────────────────────────┼──────○ (26) GPIO7
                    └─────────────────────────────┘

● = використовується для MAVLink
○ = не використовується
```

### Підключення до Matek FC

```
      Pi Zero 2 W                    Matek FC
    ┌────────────┐                ┌────────────┐
    │            │                │            │
    │  Pin 8 TXD ├───────────────►│ RX (UART)  │
    │            │                │            │
    │ Pin 10 RXD │◄───────────────┤ TX (UART)  │
    │            │                │            │
    │  Pin 6 GND ├───────────────►│ GND        │
    │            │                │            │
    └────────────┘                └────────────┘

    ⚠️ УВАГА: TX→RX, RX→TX (перехресне підключення!)
    ⚠️ НЕ підключати 5V від FC до Pi - живлення окремо!
```

### Рекомендований UART порт на Matek

| Matek FC | UART | ArduPilot Serial | Призначення |
|----------|------|------------------|-------------|
| TX3/RX3  | UART3 | SERIAL3 | MAVLink до Pi |
| TX4/RX4  | UART4 | SERIAL4 | Альтернатива |

## 3. Підключення Caddx Thermal 256 (аналогова камера)

### Характеристики Caddx Thermal 256
- Роздільність: 256×192 @ 50Hz (PAL)
- Вихід: Аналоговий PAL (CVBS)
- Живлення: 5V-20V DC
- Споживання: <1.5W

### Розпіновка Caddx Thermal 256

```
      ┌─────────────────────┐
      │  Caddx Thermal 256  │
      │     (вид ззаду)     │
      │                     │
      │  [VIN] [GND] [VIDEO]│
      │   │     │      │    │
      └───┼─────┼──────┼────┘
          │     │      │
          │     │      └──────► Жовтий (VIDEO) → EasyCap жовтий RCA
          │     └─────────────► Чорний (GND)   → EasyCap GND
          └───────────────────► Червоний (VIN) → 5V BEC від PDB
```

### Підключення через EasyCap USB Capture

```
   Caddx Thermal 256              EasyCap                Pi Zero 2 W
  ┌──────────────┐           ┌──────────────┐          ┌─────────────┐
  │              │           │              │          │             │
  │ VIDEO (жовт) ├──► RCA ──►│ VIDEO IN     │          │             │
  │              │           │   (жовтий)   │          │             │
  │ GND (чорний) ├──► GND ──►│              │   USB    │             │
  │              │           │         USB ─┼─────────►│ USB Port    │
  │ VIN (черв.)  │           │              │          │             │
  └──────┬───────┘           └──────────────┘          └─────────────┘
         │
         │ 5V від BEC
         ▼
    ┌─────────┐
    │   PDB   │ (Power Distribution Board)
    └─────────┘
```

## 4. Альтернатива: Pi Camera Module

### Підключення Pi Camera через CSI

```
      Pi Zero 2 W                    Pi Camera v2/v3
    ┌─────────────┐                 ┌───────────────┐
    │             │                 │               │
    │ ┌─────────┐ │   CSI шлейф    │ ┌───────────┐ │
    │ │   CSI   │◄├─────────────────┤►│   CSI     │ │
    │ │ роз'єм  │ │  (15-pin FFC)   │ │  роз'єм   │ │
    │ └─────────┘ │                 │ └───────────┘ │
    │             │                 │               │
    └─────────────┘                 └───────────────┘

    ⚠️ Pi Zero 2 W використовує МІНІ CSI роз'єм!
    ⚠️ Потрібен перехідний шлейф 15-pin → 22-pin
```

### Схема підключення Pi Camera

```
                    ┌─────────────────────────────────────┐
                    │                                     │
    Pi Camera v3 ───┤  Перехідний шлейф   ────► Pi Zero  │
    (22-pin CSI)    │  22-pin to 15-pin         2 W     │
                    │                        (15-pin CSI)│
                    │                                     │
                    └─────────────────────────────────────┘

    Рекомендовані шлейфи:
    - Adafruit #1648 (Pi Zero Camera Cable)
    - Довжина: 15см для мінімізації шумів
```

## 5. Живлення системи

### Схема живлення

```
    LiPo 4S (14.8V)
          │
          ▼
    ┌─────────────┐
    │     PDB     │
    │  ┌───────┐  │
    │  │ BEC 1 │──┼───► 5V/3A ──► Pi Zero 2 W
    │  │  5V   │  │              (через Micro USB або GPIO Pin 2,4)
    │  └───────┘  │
    │  ┌───────┐  │
    │  │ BEC 2 │──┼───► 5-12V ──► Caddx Thermal 256
    │  │ 5-12V │  │
    │  └───────┘  │
    │             │
    │  ┌───────┐  │
    │  │ VBAT  │──┼───► До Matek FC
    │  └───────┘  │
    └─────────────┘

    ⚠️ Використовувати окремий BEC для Pi!
    ⚠️ Мінімум 5V/2.5A для стабільної роботи
```

### Рекомендовані BEC

| Модель | Вихід | Струм | Примітка |
|--------|-------|-------|----------|
| Matek UBEC 5V | 5V | 3A | Оптимально для Pi |
| Pololu D24V50F5 | 5V | 5A | З запасом |
| Hobbywing UBEC | 5V/6V | 3A | Бюджетний варіант |

## 6. Фінальна перевірка

### Чек-лист підключень

- [ ] Pi Zero 2 W отримує стабільні 5V
- [ ] UART TX→RX, RX→TX правильно підключені
- [ ] GND Pi з'єднаний з GND Matek
- [ ] EasyCap підключений до USB Pi
- [ ] Caddx VIDEO підключений до EasyCap
- [ ] Caddx отримує живлення 5-20V
- [ ] Всі з'єднання закріплені від вібрацій

### Тест підключень

```bash
# На Pi Zero 2 W

# Перевірити USB capture
lsusb | grep -i video

# Перевірити UART
ls -la /dev/serial0

# Тест MAVLink з'єднання
python3 -c "
from pymavlink import mavutil
m = mavutil.mavlink_connection('/dev/serial0', baud=115200)
m.wait_heartbeat()
print(f'Connected to system {m.target_system}')
"
```
