# Visual Homing System для Raspberry Pi Zero 2 W + ArduPilot

## Огляд проекту

Система візуальної навігації (Visual Homing) для мультикоптерного дрона на базі ArduPilot.
Принцип роботи подібний до оптичної комп'ютерної мишки - система записує візуальні орієнтири
під час польоту і використовує їх для автономного повернення на точку зльоту.

## Компоненти системи

| Компонент | Модель | Примітка |
|-----------|--------|----------|
| Комп'ютер | Raspberry Pi Zero 2 W | 4 ядра, 1GHz, 512MB RAM |
| Термокамера | Caddx Thermal 256 (IRC-256) | Аналоговий PAL вихід |
| Альтернатива | Pi Camera Module v2/v3 | CSI інтерфейс |
| USB Capture | EasyCap (UTV007) | Аналог → USB |
| Політний контролер | Matek | ArduCopter 4.5.7 |
| Підключення | UART | MAVLink протокол |

## Особливості

- **Без GPS залежності** - навігація по візуальних орієнтирах
- **Без компаса** - орієнтація визначається візуально
- **GPS опціональний** - може використовуватись або ігноруватись
- **Teach & Repeat** - запис маршруту → повернення по записаному шляху
- **Дві версії коду** - Python (швидка розробка) та C++ (максимальна продуктивність)
- **Веб-інтерфейс** - моніторинг в реальному часі

## Швидкий старт

```bash
# 1. Клонувати репозиторій на Pi Zero 2 W
git clone https://github.com/your-repo/visual-homing.git
cd visual-homing

# 2. Встановити залежності
./scripts/install_dependencies.sh

# 3. Налаштувати автозапуск
./scripts/setup_autostart.sh

# 4. Перезавантажити
sudo reboot
```
